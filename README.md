### Bevindingen
Lijst met je bevindingen die in de test naar voren kwamen:
- De screenreader spreekt alles goed uit.
- Met de tab knop op je toetsenbord kun je gemakkelijk door mijn site heen.

#### Buttons moeten duidelijker
